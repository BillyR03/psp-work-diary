Week 7

This week we talked about testing of the product. We used the testing pyramid to structure and decide the tests we would need to use. We discussed unit tests we would need to do on functions and investigated integration testing which would need to be done when we implement back-end and front-end integration.

User interface testing and user acceptance testing had been partly done in week one and we decided we would continue this as we work. As we have decided to alter the UI and have been using emulators and an android device to view the interface, we made decisions and changes as we went and planned to continue doing this for the second sprint.
